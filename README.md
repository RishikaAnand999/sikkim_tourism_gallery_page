# Sikkim Monasteries Gallery Webpage

A prototype tourism gallery webpage showcasing the beautiful monasteries of Sikkim. The webpage provides an interactive gallery where users can explore images of monasteries with a zoomable viewing feature for better visual experience.

This project demonstrates a simple tourism-focused UI design that highlights cultural and architectural landmarks of Sikkim.

## Features

 Image gallery of famous Sikkim monasteries

 Zoom-in functionality for detailed image viewing

 Basic responsive design for different screen sizes

The gallery highlights some well-known monasteries such as:

Rumtek Monastery

Pemayangtse Monastery

Tashiding Monastery

## Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout

JavaScript – Image zoom functionality
